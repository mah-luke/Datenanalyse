Example 1: Multivariate graphics

Load the data from the file ice.RData (TUWEL) into R. These data contain in the first 8 columns the fatty acid values measured in tissue samples of human mummies. The column code classifies the observations into 6 groups, und the column group provides a more detailed description of the objects. For instance, iceman is our good old Ã–tzi, glac refers to corpses found on gaciers, and fresh refers to fresh tissue samples of recently deceased people (details in the book of Varmuza und Filzmoser, 2009).

Draw a segment and a star plot (stars()) of the first 8 columns of the ice data. Use as graphics labels the names from the column group (this column can be converted to character with the command as.character). Add a legend to the plot, and take care that the graphs, the labels and the legend are well visible.

Further, plot the first 8 columns of the ice data as parallel coordinates (function parcoord from package MASS), and use the column code to color the lines. Try to change the sorting of the variables to better reveal the data structure.

Questions:

  • Which joint properties between the mummies can you see in the plot with the segments or stars, and do these correspond to the column group? Is this better visible in the parallel coordinate plot?

R help:

  • The segment plot can be drawn with the command stars and the parameter draw.segments = TRUE. A legend can be added with the parameter key.loc.
  • A stars plot can be drawn with the same command as for segments, but the parameter draw.segments needs to be set to FALSE.
